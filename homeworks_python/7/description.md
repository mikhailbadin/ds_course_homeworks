## 3. Опимизация

Найти лучший алгоритм для решения следующей задачи:

```
min(x1x4(x1+x2+x3)+x3)
```

Условия:

```
x1x2x3x4≥30
x1^2+x2^2+x3^2+x4^2=60
2≤x1,x2,x3,x4≤6
```

Начальное приближение:

```
x=(1,5,5,1)
```
